# FT_LS Project

## Project Overview
![alt tag](https://github.com/mjbotes/readme_images/blob/master/ft_ls/Screenshot%202019-09-02%20at%2012.16.41.png?raw=true)  
FT_LS is a 42 Project that aims to mimic libc ls function

* Basic functions:
1) -l
2) -R
3) -a
4) -r
5) -t

* Bonus:
1) -1
2) -d.

## Installation
```
$> make
$> ./ft_ls {optional argument}
```
